UPLOAD GL INTERFACE INTEGRASI KE WINCORE

Petunjuk Upload GL Interface

  • created: 10/05/2020
  • latest update: 08/06/2020
  • Version 5.0
  • by: PT. Warna Bintang Kreasi

Aplikasi Interface UPLOAD GL di tujukan sebagai sarana bantu untuk mengupload data GL Bank Artos secara langsung ke Sistem Wincore

Software Minimum Requirement

  1. IIS 7+
  2. Framework ASP.NET 4.5.1
  3. Windows Server 2012 Minimum
  4. Browser (Chrome, Firefox, IE, Safari)

Hardware Minimum Requirement

    Aplication server 1 Units
  1. Processor Intel Core i7 or Higher
  2. Memori Ram 16 Gb or higher)
  3. HardDisk 500 GB or higher
    4. Client Workstation
  4. Processor Intel Core i5
  5. Memori Ram 4 Gb or higher
  6. HardDisk 300 GB or higher

Persiapan

Sebelum Instalasi setup aplikasi interface Upload GL, ada beberpa yang harus disetup :
  1. Setup Web server minimal IIS 7
  2. Jika semua sudah di setup di PC server, bisa lanjut ke instalasi setup berikutnya

Setup Web Server IIS

  1. Run box di window 10 dan ketik appwiz.cpl dan itu akan membuka menu Control Panel ; 

  2. Masuk Ke menu control panel dan pilih program and Features

  3. Klik Internet Information Services check box.

  4. Expand Internet Information Services check box dan setup seperti gambar dibawah.

  5. Klik ok dan akan muncul dialog seperti dibawah ini.

  • Ketika setup IIS sudah selesai, buka browser kemudian ketik localhost maka akan muncul seperti gambar dibawah ini.

    •  

    Setup Aplikasi UPLOAD GL Interface ke IIS Web Server

    1. Download dari Google Drive dan Extract file zip -> WBK_Web_UploadGL_V5.zip (Password zip file: wbk2020)
    2. Buat Direktory folder dengan nama "uploadgl" di folder C:\Inetpub\wwwroot\

    3. Hasil extract file dicopy paste ke folder Inetpub/wwwroot/uploadgl

    4. Buka Control panel dan pilih Internet Information Services (IIS) Manager

    5. Klik Default Website dan klik mouse kanan di Uploadgl seperti gambar berikut:

    6. Pilih Convert to Application seperti gambar berikut:

    7. Klik OK di menu pop up application pool seperti gambar berikut:

    8. Ubah Folder Akses permission satu per satu untuk folder FileGagal, FileSukses dan FileUpload ke full akses

    9. Contoh untuk Setting folder akses permission untuk folder FileGagal, klik mouse kanan pilih edit permission seperti gambar berikut:

    10. di menu properties pilih tab Security, kemudian Edit, pilh IIS_IUSRS, di list permission di kolom Allow klik Full Control, kemudian Klik OK, seperti gambar berikut:

    11. Lakukan untuk Folder lainnya untuk Akses file permission seperti langkah 9 dan 10

    12. Restart Website, klik mouse kanan di Default website, kemudian pilih manage website dan pilih Restart, seperti gambar berikut:

    Run Aplikasi Upload GL Interface

    1. Buka Browser favorit anda dan ketik localhost atau alamat web server yang sebelumnya sudah disetup aplikasi interfacenya, apabila berhasil akan muncul page login, untuk user dan password default : user: admin, password : admin

    2. ketik username : admin dan Password : admin, klik Sign in , apabila berhasil login akan muncul page Upload GL Console Dashboard

    Setting Koneksi Mesin AS400

  • Pertama kali yang harus disetup adalah parameter koneksi ke mesin AS400, Buka menu Setting, isi data source mesin wincore, user name, password dan LIbrary yang ada di mesin WINCORE, seperti contoh gambar berikut:

  • Kemudian klik Save Setting

    • Setting Schedule Upload GL Otomatis

  • Untuk Proses otomatis Upload GL ke WIncore bisa menggunakan Scheduler dan untuk setting jadwal tersebut menggunakan Cron expression seperti contoh gambar dibawah ini

  • Untuk mengetahui cara setting cron expression bisa kunjungi website https://crontab.guru/ seperti gambar dibawah ini

    • Manual Job Proses Upload

  • Menu Manual Job proses upload di tujukan untuk memproses upload data GL yang tidak mengikuti schedule atau apabila ada sesuatu proses upload data yang gagal dan bisa dilakukan proses kembali secara manual.

  • Klik OK untuk lanjut proses upload

    • Upload File Ke Server

  • File yang akan diupload harus ber extension .csv dan .done
